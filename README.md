 IPL Leaderboard Manager 🏏

A Python-based program designed to manage and update the leaderboard for a private IPL prediction/betting group with a fixed number of participants.

 📌 Overview

This project automates the process of maintaining a leaderboard for an IPL-based competition. It ensures that scores and rankings are updated efficiently as match results are entered.

The system is built for a **closed group of participants**, where the number of users remains constant throughout the tournament.

## ⚙️ Features

* 📊 Automatic leaderboard updates
* 🧮 Score calculation based on match outcomes
* 👥 Fixed participant management
* ⚡ Simple and fast execution using Python
* 📁 Easy to modify and extend

🛠️ Technologies Used

* Python
* Basic file handling / data structures

 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/skariachan/IPL-Python-program-toupdateleaderboard.git
   ```

2. Navigate to the project folder:

   ```bash
   cd IPL-Python-program-toupdateleaderboard
   ```

3. Run the program:

   ```bash
   python IPL.py
   ```
